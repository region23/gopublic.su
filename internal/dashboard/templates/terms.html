Используя сервис GoPublic, вы соглашаетесь с настоящими условиями использования. Пожалуйста, внимательно прочитайте их перед использованием сервиса.

1. Описание сервиса

GoPublic предоставляет услуги туннелирования, позволяющие временно открыть доступ к локальным сервисам через публичный интернет. Сервис предназначен для разработки, тестирования и демонстрации приложений.

2. Запрещённое использование

При использовании GoPublic строго запрещается:

  • Распространение вредоносного ПО — размещение вирусов, троянов, шпионских программ и любого другого вредоносного кода
  • Фишинг и мошенничество — создание поддельных страниц для кражи учётных данных, финансовой информации или персональных данных
  • Спам и массовые рассылки — использование туннелей для рассылки нежелательных сообщений
  • Нелегальный контент — размещение материалов, нарушающих законодательство РФ
  • DDoS-атаки — использование сервиса для организации или участия в атаках на отказ в обслуживании
  • Обход блокировок — использование сервиса для обхода законных ограничений доступа
Важно

Нарушение данных правил влечёт немедленную блокировку аккаунта без предупреждения и возможность передачи информации в правоохранительные органы.

4. Конфиденциальность

Мы храним минимально необходимую информацию для работы сервиса:

  • Идентификатор аккаунта (Telegram ID / Yandex ID)
  • Статистику использования (объём трафика, время подключений)
  • Логи для обеспечения безопасности

Мы не передаём персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством.

  • Перебои в работе сервиса

Вы используете сервис на свой страх и риск и несёте полную ответственность за контент, размещаемый через туннели.

6. Изменение условий

Мы оставляем за собой право изменять настоящие условия в любое время. Продолжая использовать сервис после изменения условий, вы соглашаетесь с новой редакцией.
